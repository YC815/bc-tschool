#### 🗺️ 停靠點詳細介紹與對應元素

1. 香山

- **對應元素**：**風**、**海**

- **景點介紹**：新竹最美的單車道起點 。擁有北台灣最大的潮間帶濕地，以巨大風車、彩虹橋與「香山夕照」聞名 。沿途融合了自然生態與海景，是感受海風與觀察濕地生物的絕佳地帶之一 。

2. 日落大道觀景台

- **對應元素**：**火**

- **景點介紹**：位於苗栗通霄落日大道，以直通大海的視覺震撼聞名 。觀景平台提供安全俯瞰點，能捕捉公路與海平面交匯的絕美景致 。尤其是黃昏時分，夕陽會精準落在道路盡頭，是攝影愛好者與單車騎士必經的熱門打卡點之一 。

3. 大甲鎮瀾宮

- **對應元素**：**土**、**木**

- **景點介紹**：台灣媽祖信仰中心，以精湛的木雕與石雕工藝聞名 。每年的「大甲媽祖遶境」為國際級宗教盛事 。廟前廣場匯聚在地信仰與傳統美食，是體驗台灣常民文化的文化核心之一 。

4. 台中舊車站

- **對應元素**：**金**

- **景點介紹**：興建於1917年的國定古蹟，採紅磚與白飾帶的「辰野式」建築風格 。它是台中城市發展的起點，現已轉型為藝文與歷史展示空間 。串聯新站與鐵道文化園區，是見證台灣鐵路百年興衰的核心地標之一 。
