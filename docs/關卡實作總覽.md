## 站一｜風 × 海

### 基本資訊
- **站別**：第 1 站 / 共 4 站
- **地點**：香山 17公里海岸線 — 南港站觀景平台
- **元素**：風、海
- **代表符號**：🌊
- **視覺主題**：藍色系（#4A90D9）

### 題目說明
**為什麼是這裡**
- 有風力發電機、海浪與沙丘（兩個元素同時存在）
- 有廁所、自行車停放區、觀景步道
- 海最具體、最安全、最好感受的地方

### 指引內容（到站時顯示）
1. 抬頭：找一支「正在轉動的風車」。
2. 聽：站在海邊聽 10 秒風聲或浪聲。
3. 看：找一條由風或海「推動」的線（旗、浪、電塔線影、草）。

### 路上任務（手機收起）
> 騎行時，只要看到「被推動」的東西，就在心裡記住。

### 回報內容（下一站才填寫）
- **照片**：一張與「流動」相關的照片
- **一句話**：記錄這趟旅途中，流動讓我＿＿＿＿。
- **字數限制**：100 字以內

### 前往下一站
- **下一個地點**：通霄 日落大道觀景台

## 站二｜火

### 基本資訊
- **站別**：第 2 站 / 共 4 站
- **地點**：通霄 日落大道觀景台
- **元素**：火
- **代表符號**：☀️
- **視覺主題**：橙色系（#E87D3E）

### 題目說明
**為什麼是這裡**
- 海面＋夕陽＋最短影子的地方
- 開闊停車區、貨櫃觀景台、簡易廁所
- 「光」在這裡是立即可見的現象

### 指引內容（到站時顯示）
1. 伸手感受太陽的熱（至少 10 秒）。
2. 看地面：尋找影子最短的瞬間。

### 路上任務（手機收起）
> 觀察一路上「光在哪裡最強」。

### 回報內容（下一站才填寫）
- **照片**：一張與「光」相關的照片
- **一句話**：光照亮了我前行的路，那一刻＿＿＿＿。
- **字數限制**：100 字以內

### 前往下一站
- **下一個地點**：大甲 鎮瀾宮前廣場

## 站三｜土 × 木

### 基本資訊
- **站別**：第 3 站 / 共 4 站
- **地點**：大甲 鎮瀾宮前廣場
- **元素**：土、木
- **代表符號**：🌳
- **視覺主題**：大地色系（#B8955A）

### 題目說明
**為什麼是這裡**
- 騎進大甲必定會遇到小坡、橋坡、堤坡（＝土）
- 廟宇木雕、老木門、梁柱木紋（＝木）
- 有廁所、飲水機、休息座椅、開放空間

### 指引內容（到站時顯示）
1. 今天第一個讓你喘的坡，停下來 10 秒。
2. 記住坡頂看到的第一樣東西。
3. 到鎮瀾宮找「最老的木紋」。

### 路上任務（手機收起）
> 留意所有木頭：老屋、門框、神轎、樑柱、木椅。

### 回報內容（下一站才填寫）
- **照片**：一張與「時間痕跡」相關的照片
- **一句話**：時間在這片土地留下的痕跡，讓我＿＿＿＿。
- **字數限制**：100 字以內

### 前往下一站
- **下一個地點**：台中舊火車站前廣場

### 基本資訊
- **站別**：第 4 站 / 共 4 站（終點）
- **地點**：台中舊火車站前廣場
- **元素**：金
- **代表符號**：🏛️
- **視覺主題**：金色系（#C9A84C）

### 題目說明
**為什麼是這裡**
- 1917 年站體，紅磚＋白線建築
- 台中流動的開端與匯聚
- 最適合「回望整趟旅程」

### 指引內容（到站時顯示）
1. 坐下來，深呼吸三次。
2. 打開相簿：從前三站挑出一張最喜歡的照片。
3. 寫一句話：「這趟旅程帶給我的＿＿＿＿。」

### 路上任務（手機收起）
> 這是終點，也是回望整趟旅程的時刻。

### 最終回報（終點填寫）
- **照片**：一張與「旅程總結」相關的照片
- **一句話**：這趟旅程帶給我的＿＿＿＿。（金的結晶）
- **字數限制**：100 字以內

### 完成旅程
- **下一步**：生成紀念卡片

## 通用頁面元素

### 頂部狀態列
- **左側**：旅者暱稱（可點擊展開選單）
  - 選單選項：捨棄旅程（需三次確認）
- **中間**：站別進度條（1-4 站視覺指示）
- **右側**：素材統計
  - 已拍攝照片數
  - 已撰寫文字數

### 進度條
- 位於最頂端
- 金色漸層（#C9A84C → #D4822A）
- 顯示目前完成進度百分比

### 頁面結構
1. **站別標題區**
   - 元素圖示
   - 站別編號（站 X / 4）
   - 元素名稱（如：風 × 海）
   - 地點名稱

2. **指引卡片**
   - 標題：「指引」
   - 編號步驟清單

3. **任務卡片**
   - 標題：「路上任務（手機收起）」或「終點任務」
   - 任務描述
   - 依站別顯示不同顏色主題

4. **回報卡片**
   - 標題：「下一站回報」或「最終回報」
   - **照片上傳區**：
     - 支援點擊上傳、拖曳上傳
     - 支援手機相機拍攝（capture="environment"）
     - 顯示預覽與移除功能
     - 格式：JPG / PNG
   - **文字輸入區**：
     - 提示文字依站別變化
     - 字數計數器（0/100）
     - 橫線稿紙視覺效果
   - **提交按鈕**：
     - 未完成時：disabled 狀態
     - 完成時：「完成，前往下一站」或「完成旅程，生成紀念卡片」

## 執行規則（給參與者）

### 一站只做四件事
1. 停
2. 開手機看指引
3. 收手機執行任務
4. 到下一站才回報

### 路上不能拍照
- 只能「到下一站」才可拍照與回報一句話。

### 全程手機只用四次
- 看指引（4 站）
- 回報（3 次）

### 每站只回報
- 一張照片＋一句話

## 技術備註

### 資料結構
```typescript
interface StationData {
  number: number;           // 站別編號
  name: string;             // 地點名稱
  elements: string[];       // 元素陣列
  emoji: string;            // 代表符號
  theme: {
    gradient: string;       // 背景漸層
    cardAccent: string;     // 卡片強調色
    cardAccentBorder: string;
    cardAccentText: string;
    cardAccentTitle: string;
  };
  guide: string[];          // 指引步驟
  task: string;             // 路上任務
  prompt: string;           // 文字輸入提示
  nextStation: string;      // 下一頁路徑
  isFinal?: boolean;        // 是否為終點
}
```

### 照片處理
- 上傳後自動壓縮與編碼為 base64
- 儲存於 IndexedDB（草稿）
- 提交時上傳至伺服器

### 資料持久化
- 草稿：IndexedDB（localStorage 備援）
- 已提交：PostgreSQL 資料庫
- 支援離線編輯，連線後同步
