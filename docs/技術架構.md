# 技術架構

## 技術棧總覽

| 層級 | 技術 | 用途 |
|------|------|------|
| 前端框架 | Next.js 15 (App Router) | React 全棧框架 |
| 語言 | TypeScript | 型別安全 |
| 樣式 | Tailwind CSS + ShadcnUI | 快速開發 UI |

| 圖片儲存 | Zeabur Storage / Cloudflare R2 | 照片上傳 |

## 資料庫結構 (Prisma Schema)

```prisma
generator client {
  provider = "prisma-client-js"
}

datasource db {
  provider = "postgresql"
  url      = env("DATABASE_URL")
}

model Player {
  id        String   @id @default(uuid())
  createdAt DateTime @default(now())
  updatedAt DateTime @updatedAt
  
  // 玩家基本資訊
  nickname String?
  
  // 關聯資料
  stations StationProgress[]
  
  @@map("players")
}

model StationProgress {
  id        String   @id @default(uuid())
  createdAt DateTime @default(now())
  updatedAt DateTime @updatedAt
  
  // 外鍵
  playerId String
  player   Player   @relation(fields: [playerId], references: [id])
  
  // 站點資訊
  stationNumber Int      // 1-4
  element      String   // wind-sea, fire, earth-wood, gold
  
  // 提交內容
  photoUrl    String   // 照片網址
  message     String   @db.VarChar(200) // 一句話（限200字）
  submittedAt DateTime @default(now())
  
  @@unique([playerId, stationNumber])
  @@map("station_progress")
}
```

## 核心功能規劃

### 1. 遊戲流程

```
首頁（掃碼進入）
  ↓
輸入暱稱 → 建立 Player
  ↓
站點 1：香山（風×海）
  - 顯示指引
  - 拍照上傳
  - 輸入：「流動讓＿＿＿＿。」
  ↓
站點 2：通霄（火）
  - 顯示指引
  - 拍照上傳
  - 輸入：「光讓＿＿＿＿。」
  ↓
站點 3：大甲（土×木）
  - 顯示指引
  - 拍照上傳
  - 輸入：「時間在這裡＿＿＿＿。」
  ↓
站點 4：台中（金）
  - 顯示指引
  - 選擇最愛照片
  - 輸入：「這趟旅程帶給我的＿＿＿＿。」
  ↓
生成紀念卡片
  - 四站照片拼貼
  - 顯示所有文字
  - 可下載分享
```

### 2. 關鍵元件

| 元件 | 功能 | 技術重點 |
|------|------|----------|
| PhotoUploader | 手機拍照/選圖上傳 | input capture="environment", 壓縮 |
| StationGuide | 站點指引顯示 | 動畫、手勢滑動 |
| TextInput | 一句話輸入 | 字數限制、即時預覽 |
| MemoryCard | 最終成果生成 | html2canvas 截圖 |

### 3. API 設計

```typescript
// POST /api/player
// 建立玩家
{ "nickname": "小明" }
→ { "id": "uuid", "nickname": "小明" }

// POST /api/upload
// 上傳照片
FormData: { file: File }
→ { "url": "https://..." }

// POST /api/station
// 提交站點內容
{
  "playerId": "uuid",
  "stationNumber": 1,
  "photoUrl": "https://...",
  "message": "流動讓我感受到自由"
}
→ { "success": true }

// GET /api/player/[id]/progress
// 取得玩家進度
→ {
  "player": { "id": "...", "nickname": "..." },
  "stations": [
    { "stationNumber": 1, "element": "wind-sea", "photoUrl": "...", "message": "..." }
  ]
}
```
